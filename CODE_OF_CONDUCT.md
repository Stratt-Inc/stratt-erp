# Code de conduite — Axiora

## Notre engagement

Nous nous engageons à faire de la participation à notre projet une expérience exempte de harcèlement pour tout le monde, quel que soit l'âge, la taille, le handicap, l'origine ethnique, l'identité de genre, le niveau d'expérience, la nationalité, l'apparence, la race, la religion ou l'orientation sexuelle.

## Standards

**Comportements acceptables :**
- Utiliser un langage accueillant et inclusif
- Respecter les différents points de vue et expériences
- Accepter gracieusement les critiques constructives
- Faire preuve d'empathie envers les autres membres

**Comportements inacceptables :**
- Harcèlement public ou privé sous quelque forme que ce soit
- Publication d'informations privées d'autrui sans permission explicite
- Tout autre comportement pouvant être raisonnablement considéré comme inapproprié

## Application

Les cas de comportement abusif peuvent être signalés via [GitHub Security Advisories](https://github.com/makcimerrr/axiora/security/advisories/new). Toutes les plaintes seront examinées et feront l'objet d'une réponse appropriée.
